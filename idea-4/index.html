Built for Developers

🌐

Browser-Based

No plugins or installations required. Edit video with standard Web Components that work in any modern browser.

⚡

Real-time Preview

See changes instantly with optimized low-resolution previews while editing. Render high-quality output on the server.

🎨

Fully Customizable

Override colors, sizes, and behaviors with CSS variables. Replace entire UI elements using slots.

🔌

Plugin Architecture

Extend functionality with custom plugins for loading, saving, exporting, and more.

🔒

REST API

Programmatic access to transcode, decode, and encode functionality via JSON-based API.

📦

Open Source

MIT licensed and available on GitHub. Contribute, fork, or use it however you need.

Quick Integration

index.html
// Import the Masher Web Component
import { Masher } from '@moviemasher/client';
// Create a new editor instance
const editor = new Masher({
container: '#editor',
theme: 'dark',
preview: true
});
// Load media and start editing
editor.load('path/to/video.mp4');
